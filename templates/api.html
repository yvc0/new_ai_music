Quick Start

Authentication

curl -X POST https://api.mixr.ai/v1/auth
  -H "Content-Type: application/json"
  -d '{"api_key": "YOUR_API_KEY"}'

Upload & Master

curl -X POST https://api.mixr.ai/v1/master
  -H "Authorization: Bearer <TOKEN>"
  -F "audio=@track.wav"
  -F "preset=professional"

Response Format

{
  "job_id": "mxr_12345",
  "status": "processing",
  "estimated_time": "8.2s",
  "webhook_url": "optional",
  "download_url": null
}
